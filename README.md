## :gear: Инструкция:

Для одноразовой обработки всех файлов можно запустить Webpack, набрав:

`npm run build`

В папке dist появятся два файла style.css и bundle.js.

Для автоматической обработки при изменении файлов можно запустить команду:

`npm run start`
